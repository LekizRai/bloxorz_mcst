# Bloxorz Solver Using Improved Monte-Carlo Search Tree

Project for Introduction to Artificial Intelligence couse


### Description
Developing new algorithm using enhanced Monte-Carlo search tree to solve many stages of famous online game Bloxorz. For more details, you can read our project's final report [here](https://github.com/LekizRai/bloxorz_mcst/blob/main/assets/project_final_report.pdf) (it is in vietnamese)

### Tools
Python, reinforcement learning, Monte-Carlo search tree
